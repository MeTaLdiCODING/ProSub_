# ProSub - Веб-платформа подписок на профессиональное ПО

**Платформа для подписки на профессиональное программное обеспечение с облачным хранилищем.**

## О проекте

**ProSub** - это SaaS-платформа, которая позволяет IT-специалистам (разработчикам, дизайнерам, монтажёрам и др.) получать доступ к профессиональному ПО по удобной модели подписки, а также использовать облачное хранилище.

### Основные возможности
- Каталог профессионального ПО
- Гибкие тарифы и пробный период
- Облачное хранилище файлов
- Личный кабинет и история платежей
- Поддержка (чат + база знаний)
- Административная панель

## Технологии
- **Frontend**: HTML5, CSS3
- **Структура**: Статический сайт
- **Уровни доступа**: 3 уровня (Гость → Пользователь → Администратор)


## Быстрый запуск

### Вариант 1: Через установщик (рекомендуется)
1. Скачайте [`ProSub Installer.msi`](./installer/ProSub%20Installer.msi)
2. Запустите файл и следуйте инструкциям установщика
3. На рабочем столе появится ярлык «ProSub - Запустить сайт»

### Вариант 2: Запуск без установки
1. Скачайте папку `site/`
2. Откройте файл `index.html` в любом современном браузере
